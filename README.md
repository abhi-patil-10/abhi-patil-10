## 🚀 Featured Projects

### 🤖 RAG-Based AI Teaching Assistant

Implemented a Retrieval-Augmented Generation (RAG) pipeline that allows users to ask:

> “Where is a specific concept explained?”

The system returns:
- 📹 Video number  
- ⏱️ Start & End timestamp  
- 📘 Context-aware explanation  

**Architecture:**  
Video → Speech-to-Text → Embeddings → Vector Database → Semantic Search → LLM → Timestamp-Based Answer  

**LLM Used:** Llama 3 (via Ollama)

🔧 Tech Used:  
`Python` `LLM` `Vector Database` `Embeddings` `RAG Architecture`

🔗 GitHub: https://lnkd.in/dv-2f73w

### 🏠 California House Price Prediction

Built an end-to-end ML pipeline to predict California housing prices.

- Data preprocessing (missing values, scaling, encoding)
- Stratified splitting for balanced train-test data
- Trained Linear Regression, Decision Tree, Random Forest
- Random Forest achieved lowest RMSE

**Production Workflow:**
- Automated model saving using `joblib`
- Skips retraining if model already exists
- Generates predictions from new data (input.csv → output.csv)

🔧 Tech Used:  
`Python` `Pandas` `NumPy` `Matplotlib` `Seaborn` `Scikit-Learn`
